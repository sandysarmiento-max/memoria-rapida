# Memoria Rápida

App sencilla para guardar recordatorios dictados por voz o escritos manualmente.

## Qué guarda

La app no guarda audios. Solo guarda el texto de la nota en el navegador usando `localStorage`.

## Archivos principales

- `index.html`
- `style.css`
- `app.js`

## Publicación en GitHub Pages

1. Crear un repositorio nuevo, por ejemplo `memoria-rapida`.
2. Subir estos archivos a la raíz del repositorio.
3. Entrar a Settings → Pages.
4. Elegir Deploy from branch.
5. Seleccionar `main` y `/root`.
6. Guardar.

La URL final será similar a:

`https://sandysarmiento-max.github.io/memoria-rapida/`
